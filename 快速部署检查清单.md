## 你需要完成的步骤

### 第一步：把代码推送到 GitHub

由于这是你的个人项目，需要你自己注册 GitHub 并创建仓库。操作如下：

1. 打开 https://github.com/signup 注册账号
2. 打开 https://github.com/new 创建新仓库
   - 名称：`daijinli-web`
   - 选择 Public
   - 不要勾选任何初始化选项
3. 创建后，复制仓库地址（如 `https://github.com/你的用户名/daijinli-web.git`）
4. 把地址发给我，或者你自己执行以下命令推送：
   ```bash
   git remote add origin https://github.com/你的用户名/daijinli-web.git
   git branch -M main
   git push -u origin main
   ```

### 第二步：获取 AI API Key

1. 打开 https://platform.deepseek.com
2. 用手机号注册/登录
3. 充值 10 元（支付宝/微信）
4. 左侧菜单 → API Keys → 创建 Key
5. 复制 Key（格式如 `sk-xxxxxxxx...`），保存好

### 第三步：创建免费数据库（Neon）

1. 打开 https://neon.tech
2. 点击 Sign Up，选择 Continue with GitHub
3. 创建项目，名称 `daijinli`，区域选 Singapore
4. 复制数据库连接字符串（Connection String），保存好

### 第四步：Vercel 部署

1. 打开 https://vercel.com/signup，用 GitHub 注册
2. 点击 Add New → Project
3. 导入 `daijinli-web` 仓库
4. **修改 Build Command** 为：`prisma generate && next build`
5. 添加环境变量：
   | 名称 | 值 |
   |---|---|
   | `DATABASE_URL` | Neon 连接字符串 |
   | `JWT_SECRET` | 随机长字符串（32位以上） |
   | `DEEPSEEK_API_KEY` | DeepSeek API Key |
   | `DEEPSEEK_BASE_URL` | `https://api.deepseek.com` |
6. 点击 Deploy

### 第五步：初始化数据库

部署成功后，访问：
```
https://你的域名/api/init-db
```
如果看到 `{"message":"数据库连接成功，表已创建"}`，就完成了。

### 第六步：测试网站

1. 访问首页，注册账号
2. 选择题型 → 出题 → 看答案 → 作答 → 批改
3. 检查历史记录是否正常保存
